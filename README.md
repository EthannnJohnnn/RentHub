# 🏠 RentHub

## 📱 Overview

**RentHub** is a mobile application developed using Kotlin that allows users to browse, manage, and book boarding houses or apartments. The system supports two types of users: **Tenants** and **Landlords**, each with their own dashboard and functionalities.

This project demonstrates the use of **Object-Oriented Programming (OOP)** and a **local database (Room/SQLite)** to manage rental listings and booking transactions.

---

## 🎯 Features

### 👤 Tenant

* View available boarding houses/apartments
* Search and filter listings
* View property details
* Book/reserve a property
* View booking status

### 🏠 Landlord

* Add new property listings
* Edit or delete listings
* Manage availability
* View booking requests
* Approve or reject bookings

---

## 🗄️ Database

The application uses a local database implemented with **Room (SQLite)**.

### Main Tables:

* Users
* Properties
* Bookings
* Reviews

---

## 🧱 Tech Stack

* **Language:** Kotlin
* **Platform:** Android
* **Database:** Room (SQLite)
* **Architecture:** MVP

---

## 🚀 How to Run

1. Clone the repository
2. Open the project in Android Studio
3. Build and run the app on an emulator or Android device

---

## 🧠 Project Purpose

The goal of this project is to simulate a **boarding house booking system** using a structured, offline database while applying core programming concepts such as OOP, database management, and mobile UI design.

---

## 📌 Notes

* This is a school project and uses a **local database only (no internet connection required)**
* Some features may still be under development
