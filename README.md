# 🏠 RentHub

## 📱 Overview
**RentHub** is a mobile application developed using Kotlin that allows users to browse, manage, and book boarding houses or apartments. The system supports two distinct types of users: **Tenants** and **Landlords**, each with their own tailored dashboard and functionalities. This project demonstrates the practical use of **Object-Oriented Programming (OOP)** and a **local database (Room/SQLite)** to manage rental listings and booking transactions.

---

## 🎯 Features

### 👤 Tenant
* View available boarding houses and apartments
* Search and filter listings
* View property details
* Book or reserve a property
* View real-time booking status

### 🏠 Landlord
* Add new property listings
* Edit or delete existing listings
* Manage property availability
* View incoming booking requests
* Approve or reject bookings

---

## 🧱 Tech Stack
* **Language:** Kotlin
* **Platform:** Android
* **Architecture:** MVP (Model-View-Presenter)
* **Database:** Room (SQLite)

---

## 🗄️ Database
The application utilizes a local database implemented with **Room (SQLite)**. 

### Main Tables:
* Users
* Properties
* Bookings
* Reviews

## 🚀 How to Run
1. Clone the repository to your local machine.
2. Open the project in Android Studio.
3. Build and run the app on an emulator or a physical Android device.

---

## 🧠 Project Purpose
The goal of this project is to simulate a functional **boarding house booking system** using a structured, offline database while practically applying core programming concepts such as OOP, database management, and mobile UI design.

---

## 📌 Notes
* This is a school project and relies entirely on a **local database** (no internet connection is required).
* Some features may still be actively under development.
